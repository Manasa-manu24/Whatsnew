# Welcome to the Whatsnew project

## Project info

This repository contains the Whatsnew real-time chat application built with Vite, React and Tailwind.

## Running the project locally

Requirements: Node.js (16+ recommended) and npm or pnpm.

Quick start:

```sh
# 1. Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

You can edit files with your preferred IDE and push changes to your remote repository.

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS

## Deployment

Deploy to your preferred hosting provider (Vercel, Netlify, Cloudflare Pages, etc.). Push your branch to GitHub and connect the repository in your hosting dashboard.
